# N2NGui

[English](./README.md)
[中文](./README.zh-CN.md)

基于N2N Edge开发的Window客户端，用于内网穿透

## 快速开始

1. 下载并解压N2NGui.zip到空文件夹下
2. 首次请以``管理员身份``启动
3. 安装网卡驱动(非必须)

![安装驱动](./docs/img/install_nic_zh.png)

4. 填写N2N启动参数

![主菜单](./docs/img/main_window_zh.png)

5. 其他选项

![其他选项](./docs/img/setting_zh.png)
